📱 Pokémon App - Flutter

Aplicación móvil desarrollada en Flutter que consume la API de Pokémon para mostrar información detallada de diferentes criaturas, incluyendo su nombre, imagen, habilidades y más.

Tecnologías utilizadas
Flutter
Dart
Dio (API REST)
Material Design
Clona el repositorio:
https://github.com/ericstevemd/pokemon.git
Entra al proyecto:
cd pokemon
Instala dependencias:
flutter pub get

Ejecuta la aplicación:
flutter run
